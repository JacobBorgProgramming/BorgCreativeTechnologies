Fruity

What is Fruity?

Fruity is a minimalist web browser designed for shared devices.

Designed for MacOS.

The Fruity Web Browser is designed to run smoothly on computers running MacOS 10.12 or newer.

FX tabs.

View the web how you want with Fruity's built-in content effects.

Built for privacy.

Fruity features a "Delete all" button to clear the browser and limited tracking. No personal data is ever recorded.
